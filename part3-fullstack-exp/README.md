# Full Stack - Gestión de Casos

Aplicación simple de gestión de expedientes con autenticación JWT.

## Tecnologías

- **Frontend:** Next.js 14 (App Router) + TypeScript
- **Backend:** Node.js + Express
- **Autenticación:** JWT (jsonwebtoken)
- **Base de datos:** En memoria (array)

## Instalación y Ejecución

### 1.Backend

```powershell
cd part3-fullstack-exp\backend
npm install
npm start
```

El backend estará corriendo en: **http://localhost:4000**

### 2.Frontend

Abre **otra terminal** y ejecuta:

```powershell
cd part3-fullstack-exp\frontend
npm install
npm run dev
```

El frontend estará corriendo en: **http://localhost:3000**
