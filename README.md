# Prueba - NXT

Este repositorio contiene las soluciones a los tres ejercicios solicitados.

## Estructura del proyecto

- **part1-books**: Aplicación Next.js que consume la API de Gutendex para mostrar un listado de libros.
- **part2-lambda**: Función serverless AWS Lambda con DynamoDB para gestión de casos.
- **part3-fullstack-exp**: Aplicación Full Stack con autenticación JWT y CRUD de expedientes.

## Tecnologías utilizadas

- Next.js con TypeScript
- Node.js y Express
- AWS Lambda y DynamoDB
- JSON Web Tokens (JWT)
- React
